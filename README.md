# Calculadora em Python com Tkinter
<h1>Descrição </h1>
<p>Esta é uma calculadora gráfica simples desenvolvida em Python utilizando a biblioteca Tkinter. A calculadora pode realizar operações aritméticas básicas como adição, subtração, multiplicação e divisão. A interface gráfica foi projetada para ser intuitiva e fácil de usar.</p>
<br>

<h1> Uso </h1>
<p> Para usar a calculadora, basta executar o script main.py:</p>
<br>
<p> Ir no CMD e aonde está o diretorio ou no Terminal da sua IDE e executar:  main.py </p>

<h1> Funcionalidades</h1>
<p> 
Adição: Permite somar dois números.
Subtração: Permite subtrair um número do outro.
Multiplicação: Permite multiplicar dois números.
Divisão: Permite dividir um número pelo outro.
Limpar: Limpa o campo de entrada para uma nova operação.
Igual: Calcula o resultado da operação.
</p>

<h1> Estrutura do Código </h1>
<p>
O código é estruturado da seguinte forma:

Importação da biblioteca Tkinter.
Configuração da janela principal da aplicação.
Definição das funções para cada operação aritmética.
Criação e disposição dos botões na interface gráfica.
</p>
